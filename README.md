## 👨‍💻 About Me

Hey, I'm **Arjun** — a full-stack developer with 3+ years of experience building scalable, production-ready web applications using **React**, **Next.js**, **Node.js**, and **MongoDB**.

I’ve engineered everything from social media platforms and eCommerce backends to custom domain SSO systems and real-time chat apps. I love working across the stack — whether it's crafting clean APIs, designing intuitive UI/UX, or deploying with **Docker** and **AWS**.

### 🔥 Highlights

* Built and deployed 10+ full-stack apps (SPA + SSR)
* Integrated features like socket.io chat, i18n, Excel uploads, and custom SSO
* Created admin dashboards, REST APIs, and reusable UI libraries
* Published [`mongo-pipeline-kit`](https://www.npmjs.com/package/mongo-pipeline-kit) — an open-source NPM package
  → [GitHub](https://github.com/arjun-computer-geek/mongo-pipeline-kit)
* Deployed apps using Docker + Nginx on AWS, Heroku, Vercel, Netlify, and Render

### 🛠 Tech Stack

React • Svelte • Next.js • TypeScript • Tailwind CSS • Node.js • MongoDB • Docker • Firebase • AWS • Heroku

### 🧑‍💼 Current & Past

* Full Stack Developer @ IIH Global
* Ex-Frontend Engineer @ Zyapaar
* NeoG Camp '22 Grad | BCA (Patna University)

### 🌍 Other

* Open to remote & product-based roles
* Open Source Contributor | Blogger
